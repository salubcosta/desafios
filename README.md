# Desafios - backend e frontend

<p>
Este repositório reúne desafios, exercícios e experimentos práticos realizados durante meus estudos de desenvolvimento backend e frontend.
</p>

<p>
O objetivo é utilizar este espaço como um laboratório de aprendizagem, onde aplico conceitos vistos em cursos, aulas e documentações, reforçando o conhecimento por meio da prática.
</p>

### Propósito

- Praticar lógica e resolução de problemas
- Fixar conceitos de programação
- Exercitar boas práticas de código
- Implementar desafios práticos

### Observação

<p>
Este repositório não faz parte do meu portfólio profissional.

Projetos de portfólio são desenvolvidos separadamente, com:

- escopo mais específico (Ex.: API Autenticação, CRUD em alguma linguáge e etc.)
- projeto com descrição pormenorizada
- foco em demonstrar habilidades reais de mercado

Aqui você encontrará principalmente exercícios menores e desafios isolados.
</p>

<p>

</p>
